# robot-qt-browser

Browser is intended to run on the RadicalES Robot-T410.

Kiosk type browser application written for Qt 5.9 LTS (C++11) targeting ARM64, dependent on Qt WebKit.

Runs on a custom ported and built version of Qt WebKit public domain version 5.212 (the last open-source release), built via Yocto. Minimum Qt compile requirement is 5.4.0.

Source is very much the same as https://github.com/qt/qtwebkit.
